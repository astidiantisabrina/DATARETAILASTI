# DATARETAILASTI

Proyek Machine Learning menggunakan metode Support Vector Classification (SVC) untuk klasifikasi kategori produk berdasarkan data transaksi retail.

## Dataset
Dataset yang digunakan:
- scanner_data.csv

## Metode
- Support Vector Classification (SVC)
- Preprocessing Data
- Feature Engineering
- One Hot Encoding
- Standard Scaling

## Library
- pandas
- numpy
- scikit-learn

## Menjalankan Program

```bash
py programretail.py
```
